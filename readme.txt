A Simple Widget to display your Twitter Wall.

== Description ==

Easy Twitter Feed Widget without creating an API to display tweets on your WordPress site.

1. NO need to create Twitter application.
1. Implement your tweets in an easy way.

Visit our Website for more: (http://www.seofixing.com/ "WP Fixes & Updates")

== Installation ==

1. Upload the "sf-simpletwitter" folder to the `/wp-content/plugins/` directory
1. Activate the SF Twitter Widget plugin
1. Drag the "SF Simple Twitter" widget to your Sidebar
1. Configure the Widget
1. Done

= How to Get the Twitter Widget ID =

1. Sign In to your `Twitter Account`
1. Go to "Settings => Widgets => Create new"
1. Click "Create Widget" button
1. You can copy the Widget ID from `Browser Address Bar` e.g. `554224441140318208`
1. OR You can copy the Widget ID from the given HTML under the Widget Preview
